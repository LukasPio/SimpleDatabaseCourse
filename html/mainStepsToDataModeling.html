Principais passos para modelagem de dados

O que é o processo de modelagem de dados?

Você já deve saber a importância da modelagem de dados, mas você sabe o como esse processo é criado, seus símbolos ou quais são os passos necessários?
O processo de modelagem de dados consiste no cumprimento de 6 passos, sendo eles:

  1. Identificação das entidades.
  2. Identificação das principais propriedades de cada entidade.
  3. Identificar relacionamentos entre entidades.
  4. Mapear atributos para entidades completamente.
  5. Normalização, redução de redundâncias e melhora de desempenho.
  6. Finalização e validação do modelo de dados.

1. Identificação das entidades.

O processo de modelagem de dados se inicia com a identificação das entidades, esse processo é responsável por identificar as entidades, itens, eventos ou conceitos presentes no sistema.

2. Identificação das principais propriedades da entidade.

Neste passo deve se identificar as propriedades das entidades, serão esses atributos que irão diferenciar umas das outras. Um exemplo disso seria a entidade carro, o carro pode ter esses atributos, que diferenciam uns dos outros:

  • Modelo
  • Cor
  • Ano de fabricação

3. Identificar relacionamentos entre entidades.

Neste passo deve se identificar as relações que as entidades tem umas com as outras, pode se pensar que os relacionamentos são como as entidades se "conversam", um exemplo disso seria: Vários carros(entidades) ficam em uma concessionária, quando um cliente(outra entidade) compra um carro ele está interagindo com o carro!

4. Identificar os atributos completamente.

Neste passo deve se identificar todos os atributos e funcionalidades do sistema. Mapeando todo o sistema será possível realizar uma padronização na modelagem de dados.

5. Normalização, redução de redundâncias e melhora de desempenho.

Neste passo deve se identificar qualquer tipo de redundância desnecessária e resolve-la, pois isso aumenta o desempenho do sistema! Mas cuidado pois isso pode melhorar ou piorar a legibilidade do sistema.

6. Finalização e validação do modelo de dados.

Esse é o último passo da modelagem de dados, nele deve se repetir os outros passo e o sistema refinado conforme o necessário.
