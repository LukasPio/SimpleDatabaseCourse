PERGUNTA 2

Quais são os tipos de modelos e suas funções?

Modelo conceitual (entendimento dos dados e suas relações) Modelo lógico (representa as relações entre dados visualmente) Modelo físico (Processo de armazenamento de dados em um sistema específico)

PERGUNTA 5

Quais são todos os elementos de um banco de dados e suas funções?

Entidade (armazena dados em linhas e colunas)
Chave primária (valor único de cada entidade- id)
Chave estrangeira (Relacionam uma entidade com a outra )
Cardinalidades (tipo de relacionamentol entre duas entidades)

PERGUNTA 6

Quais são os quatro tipos principais de cardinalidade?

Um-pra-um (1:1)
Um-pra-muitos (1:N)
Muitos-pra-muitos (N:M)
Zero-ou-Um-pra-muitos (0..1:N)

PERGUNTA 7

Quais etapas da modelagem de dados se relacionam com os modelos?

As etapas 3 e 5 (modelo lógico)
